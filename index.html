Exponential Growth in the Real World

A Complete Algebra 1 Project

1. Real-World Scenario

This project models the growth of a viral social media trend. The number of views doubles every day. This is an example of exponential growth because the rate increases over time.

2. Exponential Function Equation

The equation used is:

y = 100(2)^x

  • y = number of views
  • 100 = starting views
  • 2 = growth factor
  • x = number of days

4. Graph

5. Interpretation

The graph shows exponential growth. The number of views increases rapidly over time. The growth factor of 2 means the amount doubles each day. This demonstrates how quickly exponential functions can grow compared to linear ones.

Bonus: Linear vs Exponential

A linear model would add the same number each day, while exponential growth multiplies. This makes exponential growth much faster over time.
